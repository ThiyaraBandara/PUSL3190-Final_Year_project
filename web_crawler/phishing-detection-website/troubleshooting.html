Troubleshooting

If you encounter any issues while using the Phishing Detection System, follow these general troubleshooting steps:

  • Ensure you have the latest Python version installed.
  • Check all dependencies are installed correctly.
  • Consult error messages for resolution guidance.
  • If you encounter issues, refer to the Contact section for support.

Common Errors and How to Fix Them

  • ModuleNotFoundError: Run pip install [module_name].
  • ConnectionError: Check your network connection.
  • FileNotFoundError: Ensure the file path is correct and the file exists.
  • SyntaxError: Check for syntax issues in the code.
  • PermissionError: Ensure you have the necessary permissions.

Unique Errors Specific to This Tool

  • URLFormatError: Ensure the URL is valid and properly formatted.
  • PhishingScoreError: Check the input URL and ensure it is accessible.
  • EmailSendingError: Check your email settings and ensure the email address is correct.
  • WebScrapingError: Ensure the URL is accessible and not blocking scraping.
  • DatabaseConnectionError: Ensure your database server is running and credentials are correct.
  • InvalidEmailError: Double-check the email address format.
  • HTMLParsingError: The webpage may use non-standard HTML or heavy JavaScript.
  • WhoisLookupError: The domain may be new, private, or the WHOIS service is unavailable.
  • ReportGenerationError: Ensure you have write permissions and enough disk space.
  • TimeoutError: Check your internet connection or try a different URL.

Note: This is a static website for demonstration. Errors above are for the CLI tool, not this website.
