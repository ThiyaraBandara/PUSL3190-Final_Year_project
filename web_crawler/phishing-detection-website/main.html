About the System

In an increasingly digital world, protecting ourselves from online threats is crucial. Our Phishing Detection System is a command-line interface (CLI) tool designed to help users identify potential phishing URLs quickly and effectively.
When users enter a URL into the CLI, our system employs web scraping techniques to extract all links from that page. Each link is then analyzed for common phishing characteristics, including words like "login," "password," and "secure," as well as any suspicious symbols or numbers. The tool also checks whether links start with HTTPS, which is essential for safe browsing.
Each URL is assigned a phishing score based on these factors. The scores are averaged to determine the overall safety of the original URL, categorizing it into one of three risk levels:

  • Phishing Site: Average score of 70 or above (marked in red).
  • Maybe a Phishing Site: Average score between 31 and 69 (marked in black).
  • Not a Phishing Site: Average score of 30 or below (marked in green).

To ensure users are well-informed, the tool automatically sends a detailed report to their email address, summarizing the analysis and final safety verdict.

Our goal is to empower users with the tools they need to navigate the web safely. With the Phishing Detection System, users can feel more secure in their online activities.

How to Use

  1. Download the tool from the GitHub repository.
  2. Install required dependencies as per the documentation.
  3. Open your command line interface and navigate to the tool's directory.
  4. Run: python phishing_detection.py
  5. Follow on-screen instructions to analyze your emails or URLs.

Troubleshooting

  • Ensure you have the latest Python version installed.
  • Check all dependencies are installed correctly.
  • Consult error messages for resolution guidance.
